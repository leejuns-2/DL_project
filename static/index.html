방법론 핵심

항목정확한 표현주의할 점
Few-shot MiniLM 임베딩은 고정하고, 소수 라벨 예시로 Logistic Regression head를 학습 대형 모델 전체 fine-tuning이 아님
점수 보고서 내부 테마 간 상대 강도를 min-max 정규화 보고서 간 절대값 직접 비교 불가
수익률 연결 보고서 날짜 기준 과거 실제 수익률을 downstream 예시로 연결 미래 예측이나 투자 추천이 아님
